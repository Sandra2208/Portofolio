Halo perkenalkan nama saya Sandra Agryzal setiawan, saya merupakan mahasiswa Teknik Informatika di politeknik negeri semarang yang saat ini tertarik dalam bidang pembuatan sebuah website. Saya lahir pada tanggal 22 Agustus 2003 di Blora.

Sebelum menempuh pendidikan di Politeknik Negeri Semarang, saya bersekolah di SMA N 1 Cepu selama 3 tahun. Setelah menyelesaikan pendidikan di SMAN 1 Cepu saya tertarik dalam bidang pembuatan sebuah website yang dimana kemudian saya memutuskan untuk melanjutkan pendidikan dengan program studi Teknik Informatika. Dan saat ini saya memiliki beberapa pemahaman mengenai PHP, Laravel, HTML, css, dan MySQL
